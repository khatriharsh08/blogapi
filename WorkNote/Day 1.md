Day 1


insatll set up project by using the command
    ```composer create-project laravel/laravel blogapi```
    ```php artisan serve```

Create controllers by using the command
    ```php artisan make:controller AuthController```
    ```php artisan make:controller PostController```

Create requests by using the command
    ```php artisan make:request RegisterRequest```
    ```php artisan make:request LoginRequest```
    ```php artisan make:request StorePostRequest```

Create model by using the command
    ```php artisan make:model User```
    ```php artisan make:model Post```

Create service by using the command
    ```php artisan make:provider AuthService```
    ```php artisan make:provider PostService```

Create migration by using the command
    ```php artisan make:migration create_users_table```
    ```php artisan make:migration create_posts_table```

Run migrations by using the command
    ```php artisan migrate```

Install Api Sanctum by using the command
    ```php artisan install:api```
